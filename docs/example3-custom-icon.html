Custom Icon Integration

Learn how to replace standard map pins with fully-styled <img> elements. This approach allows for circular avatars or brand logos that respond to CSS hover states.

Implementation Code

const iconImg = document.createElement('img'); iconImg.src = 'https://images.unsplash.com/photo...'; iconImg.className = 'custom-marker-icon'; const customMarker = new AdvancedLabelMarker({ map: map, position: { lat: 49.478, lng: -123.847 }, iconHtml: iconImg, draggable: true }); customMarker.addListener('click', () => { infoWindow.open({ anchor: customMarker.marker, map: map }); });
