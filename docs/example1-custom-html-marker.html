Custom HTML Markers

Render any HTML element as a marker. This example uses a CSS Price Tag style, perfect for real estate or e-commerce applications.

Implementation Code

const labelMarker = new AdvancedLabelMarker({ map: map, position: { lat: 49.478, lng: -123.847 }, labelContent: "$425,000", labelClass: "price-tag", draggable: true, }); labelMarker.addListener("click", () => { infoWindow.open({ anchor: labelMarker.marker, map: map, }); });
